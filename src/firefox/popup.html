How to use Torrent Client

+

🔒 About the permission prompt

The first time you click Test Connection, your browser will ask permission to access your torrent client's address. This is a browser security requirement - without it, the extension cannot reach a local or remote URL outside of Nyaa.

The permission is granted only for the exact host you entered (e.g. your NAS or PC's IP), and is used solely to communicate with your torrent client. Nyaa Enhancer does not collect, transmit, or share any of your data.

  1. Select your torrent client from the Client dropdown below (qBittorrent, Transmission, or Deluge)
  2. Enter your client's URL (e.g. http://192.168.1.100:8080)
  3. Optionally fill in your credentials if your client requires authentication
  4. For qBittorrent: in the WebUI go to Options → Web UI and uncheck Enable Cross-Site Request Forgery (CSRF) protection (see note below when qBittorrent is selected)
  5. Click Test Connection to verify everything is working, then Save
  6. On any Nyaa torrent page, use the Send button to send the torrent directly to your client

Make sure Show Send to Client Button is enabled under Settings → Interface Settings, otherwise the Send button will not appear on Nyaa pages.

qBittorrent requirement

In your qBittorrent WebUI, open Options → Web UI and disable (uncheck) Enable Cross-Site Request Forgery (CSRF) protection. If this stays enabled, Sending a torrent to qBittorrent will fail.

⚠ If your server doesn't require authentication for local networks, leave the credentials fields blank.

How to monitor a user

+
  1. Visit any user's page on Nyaa (e.g., https://nyaa.si/user/username)
  2. Click the Monitor button next to their name
  3. You'll receive notifications in the sidebar to the left when they upload new torrents

The sidebar appears on the left edge of your screen and shows a notification dot when new uploads are detected.

Keyword Monitoring lets you track new uploads containing specific keywords you're interested in.

How to monitor keywords

+
  1. Visit the homepage of Nyaa or any page that shows the torrent list (e.g., https://nyaa.si)
  2. Click the Keyword Monitor button to open the monitoring popup
  3. Add a keyword you want to monitor by typing it into the input field and clicking Add Monitor
  4. You'll receive notifications in the sidebar when new torrents containing these keywords are uploaded

The sidebar appears on the left edge of your screen and shows a notification dot when new uploads are detected.

Additional Features

Screenshot Preview

Hover over a torrent link to preview its screenshot images in a floating popup. Images cycle automatically while you keep hovering.

⚠ Each hovered link sends a request to Nyaa to fetch its images. Setting the hover delay to 0 sends a request for every link your mouse passes over. Excessive requests may result in rate limiting. Use at your own risk.
